Library Project — Quick Start

Overview

Build a library system with:

Books → title, author, year, ISBN

Shelves → store books

Categories → group shelves

Client program → create objects & print output

⚠️ Output must match the expected output exactly (spacing, punctuation, capitalization, order).

| Class           | Task                                                | Test File      |
| --------------- | --------------------------------------------------- | -------------- |
| `Book`          | Complete all methods & constructors                 | `BookTest`     |
| `BookNode`      | Complete constructors                               | `BookNodeTest` |
| `BookList`      | Complete all methods & constructors                 | `BookListTest` |
| `Shelf`         | Complete all methods & constructors                 | `ShelfTest`    |
| `Category`      | Complete all methods & constructors                 | `CategoryTest` |
| `LibraryClient` | Create objects & call methods to match exact output | N/A            |


Complete each class in the order above.

Run its JUnit test before moving to the next class.

Once all classes pass, complete LibraryClient.java to produce exact output.

Submit all six .java files.
